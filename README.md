# 💰 FinWise — Controle Financeiro

> Controle financeiro simples, inteligente e visual para acompanhar gastos, receitas e metas financeiras.

🔗 **Acesse o app:** [fin-wise-scope.lovable.app](https://fin-wise-scope.lovable.app)

---

## 📱 Sobre o Projeto

O **FinWise** é uma aplicação web de controle financeiro pessoal, desenvolvida para ajudar usuários a gerenciar suas finanças de forma simples e visual. Com dashboard interativo, registro de transações, metas financeiras e acompanhamento de cotações, o app oferece uma visão completa da vida financeira do usuário em um único lugar.

---

## ✨ Funcionalidades

- 🔐 **Autenticação** — Login via Google, Apple ou email/senha
- 📊 **Dashboard** — Visão geral com total de entradas, saídas, saldo, gasto médio diário e maior categoria de despesa
- 💳 **Transações** — Cadastro de entradas e despesas com categorização
- 📈 **Relatórios** — Análises detalhadas com filtros por período (7 dias, 30 dias, total)
- 🎯 **Metas** — Definição e acompanhamento de metas financeiras
- 🏦 **Bancos** — Gestão de contas e instituições bancárias
- 💡 **TipsMoney** — Conteúdo educativo sobre investimentos (Renda Fixa, Renda Variável, Criptomoedas, Reserva de Emergência)
- 💱 **Cotações** — Acompanhamento de moedas em tempo real (Dólar, Euro, Libra, Bitcoin e mais)
- 💬 **Feedback** — Canal direto para sugestões dos usuários
- 👤 **Perfil e Configurações** — Personalização da conta

---

## 🎨 Design

- **Tema:** Dark mode moderno
- **Paleta de cores:**
  - 🟢 Verde — Entradas / valores positivos
  - 🔴 Vermelho — Saídas / valores negativos
  - 🔵 Azul — Informações neutras
  - 🟣 Roxo — Categorias e destaques
- **UI:** Cards arredondados, navegação lateral, foco em legibilidade e usabilidade

---

## 🛠️ Tecnologias

Este projeto foi desenvolvido utilizando [Lovable](https://lovable.dev), uma plataforma de desenvolvimento assistido por IA, com:

- **React** + **TypeScript**
- **Tailwind CSS**
- **Vite**

## 🚀 Como Executar Localmente

```bash
# Clone o repositório
git clone <URL_DO_REPOSITORIO>

# Acesse a pasta do projeto
cd finwise

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```
